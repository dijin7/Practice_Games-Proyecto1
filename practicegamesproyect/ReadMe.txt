Ignacio Silva Novo - Entrega proyecto 2

Problemática:
Se quiere, dentro de todo, hacer un programa en el cual se busca
poder dar acceso al usuario a ciertos juegos para practicar rutinas. 
Cada rutina tiene una serie de pasos en el cual se quiere guardar el progreso.

Funciones esperadas:
- Juegos con distintas rutinas
- Rutinas que se les pueda guardar el progreso
- Historial
- Menú desplegable
